### 差异化定价机会

所有竞品要么全付费墙、要么靠内容广告变现。**本项目是唯一"工具全部免费、靠体验升级收费"的模型**——这本身就是一个定价差异化。

## 定价模型设计

### MVP（Stage 07 上线）：Free Only

```
Free 层：一切免费
  ├── 4 种呼吸模式（含自定义）
  ├── 完整 15 题倦怠评估 + 结果
  ├── 数字排毒计划生成
  ├── 无需注册
  └── 无广告（MVP 期）

不设付费墙。
先用免费换流量、换外链、换品牌认知。
```

### 为什么 MVP 不接支付

| 理由 | 说明 |
|------|------|
| **流量不够** | 月访问 < 1000 时，转化率 0.5-2% = 5-20 个付费用户，不值得上线支付系统的复杂度 |
| **免费是护城河** | 竞品都有付费墙，全免费 = 差异化获客 |
| **先验证需求** | 用 GSC + GA 数据看哪个工具使用率最高，再决定 Pro 卖什么 |
| **合规成本** | 支付 = PCI + 退款政策 + 税务 + 更多法律页面，MVP 不承载 |

### Pro 模型（Phase 3+ 引入，当 DAU > 200）

```
Pro 层：$4.99/月 或 $39/年（≈$3.25/月）

  ├── 保存自定义呼吸模式（最多 5 组）
  ├── 倦怠评估历史记录（追踪趋势）
  ├── 数字排毒计划保存 + 进度追踪
  ├── 去广告
  └── 优先体验新工具

为什么 $4.99 而不是 $9.99：
  - 低于所有竞品（Calm $14.99 / Insight Timer $9.99）
  - 不要和冥想 App 比——你是工具站，不是内容订阅
  - $4.99 对职场人是"不犹豫"价格（< 一杯咖啡/月）
  - 留涨价空间（后期可到 $7.99）
```

### 为什么不卖 Lifetime

```
❌ Lifetime：工具站持续运营成本（域名/服务器/更新）+ 无限支持承诺 ≠ 一次性收费模型
✅ 年付折扣：$39/年 给用户省钱、给你可预测收入
```

## 下游交接摘要

### 当前结论
- **状态**: [DONE]

### 核心判断
- MVP Free only，不接支付
